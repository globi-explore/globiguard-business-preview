# GlobiGuard strategy preview

Temporary read-only business preview site for sharing the `business/` strategy pack as a polished website.

## Notes

- built with Docusaurus
- intended for short-lived GitHub Pages sharing
- includes a lightweight client-side password gate only
- includes `noindex` and `robots.txt` deterrents, not real protection

## Local development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## GitHub Pages

The repo includes `.github/workflows/deploy.yml` for GitHub Pages deployment via Actions.
